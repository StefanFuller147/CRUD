# CRUD
Our week 7 project at Skill Distillery, known as a CRUD project (Create, Read, Update, Delete) 
the purpose of this project was to be able to create, read, update, and delete information from a website. 
## Application description
Here the user is taken to a main index of all current Rainbow 6 operators. Here they can view, edit, or delte all 20 current operators.
Cycle through all operators, and view their videos. From the main index page, you can create a brand new operator. 

### Technologies used
*Java
*Spring
*HTML
*Tomcat8
